   - Go to the **General** tab
   - Use the **Language Switcher** to select your preferred language

## Importing Your First Book

1. Click **"Import EPUB"** button in the top navigation
2. Select an EPUB, PDF, TXT, Markdown, or HTML file
3. Wait for the import to complete
4. Your book will appear in the library

## Reading a Book

1. Click on a book card in the library
2. The book opens in the immersive reader
3. Use the chapter list on the left to navigate
4. Press **F11** to enter fullscreen reading mode

## Taking Notes

1. While reading, select any text
2. Click **"Highlight"** or **"Annotation"** in the popup menu
3. Click **"Create Note"** to add a detailed note
4. Access your notes from the Notes sidebar

## Using AI Assistant

1. Go to **Settings** ‚Üí **AI Assistant Configuration**
2. Enter your API key (OpenAI, Anthropic, or Gemini)
3. Select text while reading
4. Click **"AI Explain"** to get AI-powered explanations

## Keyboard Shortcuts

- **F11**: Toggle fullscreen reading mode
- **Esc**: Exit fullscreen mode
- **Arrow Keys**: Navigate between chapters (when available)

3. **Smart Chapters**: The app automatically merges fragmented chapters for better reading flow.
4. **Progress Tracking**: Your reading position is automatically saved.
